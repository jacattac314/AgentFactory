# Watch My Screen And Press Accept On Any Agent

> Auto-generated by Hermes Agent Factory · template: `computer_use_agent`

## What it does

Watches your screen on a configurable poll interval. Each cycle it captures a
screenshot, asks Claude to check whether a Codex confirmation pop-up is visible,
and clicks the Accept button if one is found. It stops when Codex reports
completion or the maximum cycle count is reached.

## Configuration

Pass a `config` dict to `run()`:

| Key | Default | Description |
|-----|---------|-------------|
| `poll_interval_seconds` | `3` | Seconds between screenshots |
| `max_iterations` | `300` | Maximum polling cycles (~15 min at 3 s) |
